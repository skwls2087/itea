벤치마킹 : https://jasoseol.com/recruit

주제와 목적

1. IT 자격증 시험정보와 일정을 제공
2. IT 자격증 문제를 출제하고 풀어보는 서비스 제공
3. 회원들이 IT 정보를 자유롭게 공유하는 게시판과 실시간 채팅 서비스 제공
3. 직업 만족도 테스트 제공
4. 월별 IT 뉴스 워드클라우드 제공
5. 개발자 관련 통계자료 제공

  9. 형상화 도구 : Git
